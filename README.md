# Meraki Travel — a personal travel companion & journal

A mobile-first web app that turns a trip into a swipeable, map-based, voice-guided guide — and quietly becomes a journal of everywhere you go. Built first for Paris; designed to grow city by city.

**Live app:** _(Vercel URL added on first deploy)_

---

## What it does today

- **Discover** — swipe through hand-picked places (food, cafés, art, music, photo spots, local gems, experiences). Swipe right to save, left to skip.
- **Stories** — tap *Listen* and a voice reads the history + a hidden fact about each place.
- **Near Me** — GPS sorts everything by how close it is, live as you walk.
- **Map** — every place as a colour-coded pin with details + directions.
- **Journal** — when you arrive at a saved place, the app asks how it felt. Write a line, pick a mood. Export the whole trip as a file (`Export file`) that can be turned into a printed journal later.

## How to run locally

```bash
cd app && python3 -m http.server 8741
# open http://localhost:8741
```

The deployable file is `app/index.html`. It is **generated** — don't hand-edit it.

## How to edit / rebuild

1. Edit content/data in `data/paris-places.json` or the source UI in `app/template.html`.
2. Rebuild: `python3 build.py`
3. Commit & push — Vercel auto-deploys (once Git is connected, see below).

### Changing names & wording
For now the personal touches ("Made for Ramya", "Meraki Paris Tour", category labels) live in `app/template.html` near the top of the `<body>` and in the `CATS` object. A `config.js` to centralise these is on the roadmap.

## Project structure

```
.
├── app/
│   ├── index.html        # deployable app (generated by build.py)
│   └── template.html     # source: UI + logic, with /*__PLACES_DATA__*/ slot
├── data/
│   ├── paris-places.json # master place list (the live data)
│   └── cities/
│       └── paris.json    # city-pack copy (seed for the multi-city framework)
├── build.py              # injects data into the template
├── vercel.json           # static hosting config (serves app/)
├── ROADMAP.md            # where this is going
└── PLACES.md             # human-readable list of all places
```

- One-off: `cd app && npx vercel deploy --prod`
